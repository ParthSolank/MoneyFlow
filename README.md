# MoneyFlow Pro

A modern personal finance management application built with Next.js and .NET.

## 🚀 Make It Your Own
I have included a detailed guide on how to customize and deploy this app as your own.
Check out: **[docs/make-it-your-own.md](./docs/make-it-your-own.md)**

## Features
- **Financial Overview**: Real-time dashboard tracked in Indian Rupees (₹).
- **Ledger Management**: Full control over bank accounts and credit cards.
- **Transaction Tracking**: Detailed history with categories.
- **Role-Based Access**: Specialized views for Admin and Standard Users.

## Tech Stack
- **Framework**: [Next.js 15](https://nextjs.org/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Backend**: [.NET 8 Web API](https://dotnet.microsoft.com/)
- **Database**: [SQL Server](https://www.microsoft.com/en-us/sql-server/)

## Local Development
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:9002](http://localhost:9002) in your browser.
