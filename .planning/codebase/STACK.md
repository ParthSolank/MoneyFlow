# Stack: MoneyFlow Pro

## Core Technologies
- **Frontend**: Next.js 15, Tailwind CSS, Lucide icons, Framer Motion
- **Backend**: ASP.NET Core 8.0 (C#)
- **Database**: SQL Server (LocalDB / Express)
- **Data Access**: Entity Framework Core
- **Client-side State**: SWR, Context API
- **Authentication**: JWT (JSON Web Tokens)
- **Date Handling**: Native Date objects, `financial-year-utils`

## Key Dependencies
- `csvhelper`: For CSV parsing (backend)
- `exceldatareader`: For Excel parsing (backend)
- `questpdf`: For PDF generation (backend)
- `radix-ui`: UI components (frontend)
- `shadcn/ui`: Component library (frontend)

## Runtime Requirements
- Node.js 18+
- .NET 8.0 SDK
- SQL Server LocalDB or SQLEXPRESS
